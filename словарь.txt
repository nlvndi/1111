  Чтобы авторизироваться вводим команды :
   git config --global user.name "ваше имя в гитхаб"
   git config --global user.email "ваша почта"
Команда pwd - выводит пууть к текущему рабочему каталогу
Чтобы создать новый репозиторий используется комнада:git init
git commit -m "___" - фиксация изменений и записи их в репозиторий.
 ls-la - показывает все файлы в директории.
 git status - сотояние рабочего каталога.
Чтобы сохранить изменения используется команда: git add .
 git remote -v - показывает удаленные репозитории
 git restore - команда чтобы отменить изменения в рабочем каталоге.
 git blame <имя> - эта команда позволяет изучить определенные этапы изменений файла и узнать, кто внес последние изменения в файл
git restore <имя файла> - отменить изменения в опр.файле
 Команда git log - показывает историю коммитов.
git push - используется для выгрузки содержимого локального репозитория в удаленный.
Команда git pull - извлекает и загружает содержимое из удаленного репозитория в локальный.
Чтобы создать доп.ветку:git checkout -b branch1
Что бы удалить из индекса некоторые файлы используйте :git restore --staged <имя файла> 
gitignore - игнорирование файлов.
git diff - выводит изменения всех файлов.
git merge - выполняет слияние одной или нескольких веток в  текущую.
git rm - удаление файлов из индекса и рабочей копии.
 git branch - "менеджер веток" перечисляет ветки, создает новые, удаляет и переименовывает их.
 git checkout - позволяет переключиться между ветками.
 git stash - используется для временного сохранения всех незакоммиченных изменений для очистки рабочей директории без необходимости коммитеть незавершенную работу в новую ветку.
git remote - служит для управления списком удаленных репозиториев.
git show - отображает объект в простом и понятном виде.
git stash pop - выводит скрытые временные изменения.
